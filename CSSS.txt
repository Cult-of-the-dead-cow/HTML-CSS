CSS
------------
	stands for Cascading stylesheet.
	used for web designing.
	
	Types
	---------
	1)Inline stylesheet	
	2)Internal Stylesheet
	3)External Stylesheet
	
	1)Inline Stylesheet
	---------------------------
		we can use style property for applying different styles.
		Properties are separated by semicolon(;) and assign by using colon(:).
	
		<tagname style="">text</tagname>

		
	color:red;

	Font properties
	------------------------
	font-family:"name of the font";
	font-size:12px;
	text-decoration:underline/overline/line-through;
	text-transform:uppercase/lowercase/capitalize;
	word-spacing:10px;
	letter-spacing:10px;
	line-spacing:10px;
	text-align:left/right/center/justify;

	<h1 style="font-family:"jokerman";font-size:12px;text-decoration:uppercase">welcome</h1>
